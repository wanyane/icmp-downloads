Getting it running

No Python, no installer

Windows

Unzip, open the icmp folder, double-click icmp.exe. Keep the other files alongside it.

macOS

Right-click the app → Open the first time (Gatekeeper blocks unsigned apps opened by double-click).

Linux

Enter the icmp folder, chmod +x icmp if needed, then run ./icmp.

Using it

Pick two images, compare them

  1. Click Open Folder… in either explorer, or type/paste a path (autocomplete helps) and press Enter.
  2. Click an image to select it — first click becomes Left, second becomes Right, from either explorer.
  3. Press C to open the Compare page. Scroll to zoom, drag to pan — both sides stay in sync.
  4. Hold Overlay (or Space) to swap Left for Right in place; release to go back.
  5. Press Shift+? any time for the full shortcuts help.
